# vue_dz


Vue 3 + Vite

> Vite - https://vite.dev/guide/


## Создание проекта:
```bash
npm create vite@latest
```

Переходим в директорию проекта:
```bash
cd vite-project
```

## Установка:
```bash
npm install
```

## Запуск дев сервера
```bash
npm run dev
```

## Запуск unit-тестов
```bash
npm run test
```

## Запуск e2e-тестов
```bash
npm run e2e
```
